# NitroTools
Some tools for Nitro Sound Data (SDAT)

## Included Tools
* SymbTool - An extractor and packer for symb.bin filenames.

## Planned Tools
* InfoTool - An extractor and packer for info.bin track info.
* Nitro Studio - GUI editor for extracted sdats.
